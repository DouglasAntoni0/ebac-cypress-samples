Repositório de estudos com exemplos Cypress, organizado para praticar comandos, fixtures, suporte e cenários de automação end-to-end.

## O que este projeto demonstra

| Área | Evidência no projeto |
| --- | --- |
| Cypress básico | Estrutura padrão com `cypress/e2e`, `fixtures` e `support` |
| Testes exploratórios guiados | Exemplos de comandos e padrões do framework |
| Execução automatizada | Script `npm test` preparado para rodar `npx cypress run` |
| Base de aprendizado | Exercícios que sustentam projetos Cypress mais completos neste perfil |

## Estrutura

```text
.
├── cypress/
│   ├── e2e/
│   ├── fixtures/
│   └── support/
├── cypress.config.js
├── package.json
└── package-lock.json
```

## Como executar

```bash
git clone https://github.com/DouglasAntoni0/ebac-cypress-samples.git
cd ebac-cypress-samples
npm install
npm test
```

Para abrir a interface do Cypress:

```bash
npx cypress open
```

## Resultado técnico

O repositório mostra familiaridade com a anatomia do Cypress e seus recursos de apoio. Essa base é importante porque bons projetos de QA começam com domínio do framework antes de avançar para arquitetura, relatórios, CI/CD e padrões de manutenção.
